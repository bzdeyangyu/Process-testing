HTTP Server 端口 8765；支持 CORS（Access-Control-Allow-Origin: *）；所有响应 Content-Type: application/json 或 text/html
MethodPath功能说明返回
GET /boards/current 返回最新工程看板快照；图片 local_path 自动转为 /renders/ HTTP URL project_board_current.json
GET /boards 返回所有历史运行记录列表（project_board_index.json） { runs: [...] }
GET /renders/<path> 提供生成图片的静态访问；支持子目录（scheme_A/render_1.png）；有路径穿越保护 image/png 等
GET /status 查询当前是否有工作流在运行 { running: bool, last_result_keys: [...] }
GET / 重定向到 dashboard/index.html；提供看板 SPA text/html
POST /run 在后台线程异步启动 MVP 工作流；若已在运行返回 409；支持 brief / image_provider / image_api_key 参数 202: { status: "started", brief, image_provider }
